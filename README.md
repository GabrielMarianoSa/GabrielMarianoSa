## Sobre

Sou desenvolvedor full stack júnior com experiência em front-end e back-end, atuando com foco em resolver problemas reais através de código limpo, boas práticas e interfaces intuitivas.

Tenho forte atuação no ecossistema JavaScript (React, Node.js, TypeScript) e também no universo TOTVS, utilizando ADVPL, TLPP e SQL em projetos corporativos.

Atualmente trabalho com desenvolvimento em consultoria TOTVS e projetos web.
